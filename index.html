    Esercizi di Programmazione¶

    Benvenuto nella raccolta didattica di esercizi di programmazione pensata per gli studenti della scuola secondaria di secondo grado.

    Qui trovi esercizi pratici, ordinati per linguaggio o argomento, con spiegazioni chiare e la possibilità di provare il codice direttamente dal browser grazie a OneCompiler.

    Le sezioni disponibili¶

    Sezione Argomento Livello indicativo
    Bash Script da riga di comando su Linux/macOS Base → Intermedio
    C Linguaggio C: variabili, cicli, funzioni, puntatori Base → Avanzato
    Database SQL: query, JOIN, DDL e DML su database relazionali Base → Intermedio
    Java Programmazione orientata agli oggetti in Java Intermedio → Avanzato
    PacketTracer Reti con Cisco Packet Tracer: LAN, IP, router Base → Intermedio
    PHP PHP lato server: variabili, form, sessioni Base → Intermedio
    Python Python: dalla sintassi base a funzioni e file Base → Avanzato

    Come funziona il sito¶

    1. Scegli una sezione dal menu in alto o dalla tabella sopra.
    2. Leggi la descrizione dell'esercizio e analizza il codice di esempio.
    3. Prova il codice con il pulsante OneCompiler incorporato nella pagina, oppure copialo nel tuo ambiente di sviluppo locale.
    4. Consulta la soluzione solo dopo aver provato da solo — imparare significa sbagliare e correggere!

    Suggerimento

    Se sei alle prime armi, inizia dalla sezione Python o Bash: sono i linguaggi con la sintassi più immediata.

    Leggi la pagina Come usare il sito per scoprire come navigare, eseguire il codice online e scaricare i file degli esercizi.
