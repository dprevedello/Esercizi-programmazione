# :simple-gnubash: Bash

**Bash** (Bourne Again SHell) è la shell predefinita di Linux e macOS.
Scrivere script Bash significa automatizzare operazioni da riga di comando:
copiare file, elaborare testo, eseguire programmi in sequenza e molto altro.

È un ottimo punto di partenza per capire come funziona il sistema operativo
"dietro le quinte".

---

## Cosa imparerai

- Eseguire comandi fondamentali del terminale (`ls`, `cd`, `cp`, `mv`, `rm`…)
- Scrivere script con variabili, cicli (`for`, `while`) e condizioni (`if`)
- Lavorare con file di testo: leggere, filtrare e trasformare l'output
- Usare pipe (`|`) e redirezione (`>`, `>>`, `<`)
- Scrivere funzioni riutilizzabili all'interno di uno script
