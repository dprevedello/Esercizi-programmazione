# :material-language-python: Python

**Python** è uno dei linguaggi più popolari al mondo: è chiaro, leggibile e
versatile. Si usa per analisi dati, intelligenza artificiale, automazione,
sviluppo web, scripting e molto altro.

È spesso il primo linguaggio che si studia proprio perché la sua sintassi è
vicina al linguaggio naturale.

---

## Cosa imparerai

- Variabili, tipi di dato e operatori
- Input/output: `print()` e `input()`
- Strutture di controllo: `if`, `elif`, `else`, cicli `for` e `while`
- Liste, tuple, dizionari e insiemi
- Funzioni: definizione, parametri, valori di ritorno
- Scope e funzioni lambda
- Moduli della libreria standard (`math`, `random`, `os`, `datetime`…)
- Lettura e scrittura di file
- Gestione delle eccezioni (`try`, `except`, `finally`)
- Cenni alla programmazione orientata agli oggetti (classi e oggetti)

## Eseguire Python in locale

```bash
# Verifica la versione installata
python3 --version

# Esegui uno script
python3 esercizio.py

# Avvia la shell interattiva (REPL)
python3
```

!!! tip "Editor consigliato per chi inizia"
    **Thonny** ([thonny.org](https://thonny.org)) è un editor gratuito
    pensato per chi impara Python: mostra i valori delle variabili passo
    dopo passo.
