# Database e SQL

Un **database relazionale** organizza i dati in tabelle collegate tra loro.
Il linguaggio **SQL** (Structured Query Language) permette di interrogare,
inserire, modificare ed eliminare questi dati in modo strutturato e preciso.

SQL è usato ovunque: app web, gestionali, e-commerce, sistemi bancari.
Conoscerlo è una competenza molto richiesta nel mondo del lavoro.

---

## Cosa imparerai

### SQL — Query e dati

- `SELECT`, `WHERE`, `ORDER BY`, `LIMIT` — leggere i dati
- `JOIN` (INNER, LEFT, RIGHT) — collegare tabelle
- Funzioni di aggregazione: `COUNT`, `SUM`, `AVG`, `MAX`, `MIN`
- `GROUP BY` e `HAVING`
- Sottoquery e query nidificate
- DDL: `CREATE TABLE`, `ALTER TABLE`, `DROP TABLE`
- DML: `INSERT`, `UPDATE`, `DELETE`
- Chiavi primarie, chiavi esterne e vincoli

### Algebra relazionale

- Selezione (σ), proiezione (π) e ridenominazione (ρ)
- Prodotto cartesiano e join naturale
- Unione, intersezione e differenza
